# Rhythmix
Rhythmix is a rhythm game made in Unity for the final year project. <br>

In this game there are two circular play areas where notes will spawn in the middle. These notes will move to the outside of the play area and they player must time their inputs with a controller to hit these notes. There are tap notes where the player only needs to press down and release and hold notes where the player must press and hold when the head reaches the hit area and release when the tail reaches the play area. Hold notes have a line connecting the head and the tail so you can tell the difference between them and a tap note. <br>

You can also compete against other players with the gloabl leaderboards.

There is also a level editor where players can add their own music and create their own levels. They must also enter the BPM of the song and how they would like the beats to be divided. 

## Installation
To play the game download the latest release found here [https://github.com/Thefudgeman/COMP3000/releases](https://github.com/Thefudgeman/COMP3000/releases) and extract all in file explorer<br>

Before playing the game open the folder which will contain a folder called DefaulLevel. Copy the folder within this to AppData/LocalLow/DefaultCompany/COMP3000/Music. This will add the pre-made level into the game. <br>

To open the game run the executable file called COMP3000 within the downloaded folder

## Controls/How To Play
Never make any maual changes to the names or contents of the text files within the Music folder mentioned earlier unless you are adding new levels. Any changes made will cause the game to be unable to find the song, change the notes in the level or cause the level to be unplayable.

All menus and UI must be interacted with a mouse or keyboard <br>

Within the gameplay use a controller. The D-pad is used for the left circle and the ABXY buttons on an Xbox controller is used for the right circle. To pause the level press the menu button on the controller or escape on the keyboard. <br>

In the level editor you can use the left or right arrow keys to go through the song or use the play/pause button in the top left. To swap between adding hold notes and tap notes press H on the keyboard. When adding hold notes both the head and tail must be added. If you want your level to be addded to the leaderboards go to the main menu and click on the submit button. Then navigate to the folder created for your level and click on the text file which contains all the information about the map. This will create a leaderboardID so you can share it and compete against others.
